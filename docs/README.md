## Requisitos

- Kali Linux o sistema compatible con herramientas de pentesting.
- Acceso a red local con las máquinas vulnerables.
- Cliente SSH, nmap, gobuster, etc.

## Notas importantes

- Todas las máquinas están diseñadas para uso educativo.
